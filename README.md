# Step Final Project

This repository contains the source code for a Godot-based game project.

## Overview

The project is a 2D adventure game with scenes, scripts, UI, combat systems, inventory interactions, dialogue, and level progression. The source code is organized into:

- scripts/: gameplay logic, player controls, combat, UI, and scene management
- world/: Godot scenes and game levels
- resources/: game assets and media files

## Project Structure

- scripts/ - core game logic and systems
- world/ - scene files and level layouts
- project.godot - Godot project configuration

## Notes

This project uses Godot and includes asset resources and imported media files. The repository is configured to ignore large asset folders and image-based resources where appropriate.

## Getting Started

1. Open the project folder in Godot.
2. Load project.godot to open the project.
3. Run the game from the editor to play and test.
